# HisaabKitaab 📒

Your personal expense splitting app — track who paid what, and who owes whom.

## Project Structure

```
MySplitwise/
├── hisaabkitaab/
│   ├── models/      # Database models (User, Group, Expense, etc.)
│   ├── services/    # Business logic (splitting, debt simplification)
│   ├── api/         # FastAPI routes
│   ├── cli/         # CLI interface
│   └── tests/       # Unit & integration tests
├── requirements.txt
└── main.py
```

## Setup

```bash
pip install -r requirements.txt
python main.py
```
